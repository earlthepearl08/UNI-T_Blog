## 1. Brand Overview

UNI-T is a leading manufacturer of electronic test and measurement equipment. The UNI-T Philippines brand presence should convey:

- **Professional expertise** in electrical testing
- **Reliability** and precision
- **Accessibility** for both professionals and hobbyists
- **Modern** technical sophistication

## 4. Typography

### Font Stack

```css
/* Headlines */
font-family: 'Segoe UI', Roboto, -apple-system, BlinkMacSystemFont, sans-serif;
font-weight: 700; /* Bold */

/* Body Text */
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;
font-weight: 400; /* Regular */
```

### Type Scale

| Element | Size | Weight | Line Height |
|---------|------|--------|-------------|
| H1 | 32px | Bold (700) | 1.2 |
| H2 | 24px | SemiBold (600) | 1.3 |
| H3 | 20px | SemiBold (600) | 1.4 |
| Body | 16px | Regular (400) | 1.6 |
| Caption | 14px | Regular (400) | 1.5 |
| Small | 12px | Regular (400) | 1.5 |

## 5. Imagery Style

### Photography Guidelines

- **Subject**: Test equipment, workbenches, technicians at work
- **Lighting**: Well-lit, professional
- **Composition**: Clean, uncluttered backgrounds
- **Color**: Natural colors, can have subtle red accent lighting
- **Quality**: High resolution, sharp focus on products

### Icon Style

- Line icons preferred
- 2px stroke weight
- Rounded corners
- Single color (adapt to context)
- Technical/electrical theme

### Illustration Style

- Clean, minimal vector style
- Limited color palette
- Technical accuracy
- Isometric view for equipment when possible

## 7. Voice & Tone

### Brand Voice

- **Knowledgeable** but not condescending
- **Professional** but approachable
- **Helpful** and practical
- **Confident** in expertise

### Writing Guidelines

- Use active voice
- Keep sentences concise
- Explain technical terms when necessary
- Include actionable tips
- Avoid jargon without explanation
