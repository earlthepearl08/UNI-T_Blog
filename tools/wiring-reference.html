Circuit Breaker Size Guide (Philippine)

Wire size to breaker size mapping based on Philippine Electrical Code standards.

Wire Size (mm²) Max Breaker (A) Typical Use
1.5 mm²15ALighting circuits
2.5 mm²20AGeneral outlets / convenience receptacles
4.0 mm²25AHeavy-duty outlets, small air-conditioning units
6.0 mm²32ALarge AC units, electric water heaters
10 mm²50ASub-panel feeds, electric ranges
16 mm²63AMain panel feed
25 mm²80ALarge main feed / service entrance
Pro Tip: Always verify wire gauge with a UNI-T UT61E+ multimeter (continuity and resistance check) before connecting to a breaker. Use a UT210E clamp meter to confirm actual load current.
