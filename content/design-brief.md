## Brand Identity

### Primary Colors
| Color | Hex | RGB | Usage |
|-------|-----|-----|-------|
| **UNI-T Red** | #E31E24 | 227, 30, 36 | Primary accent, CTAs, highlights |
| **UNI-T Red Dark** | #B8181D | 184, 24, 29 | Hover states, depth |
| **Black** | #1A1A1A | 26, 26, 26 | Text, headers, backgrounds |
| **Dark Gray** | #333333 | 51, 51, 51 | Secondary text |
| **Light Gray** | #F5F5F5 | 245, 245, 245 | Backgrounds, cards |
| **White** | #FFFFFF | 255, 255, 255 | Primary background |

### Typography
- **Headlines**: Segoe UI, Roboto, or similar sans-serif (Bold/SemiBold)
- **Body Text**: System fonts (-apple-system, Roboto)
- **Style**: Clean, professional, technical but approachable

### Visual Style
- Clean, minimalist design
- Professional and technical feel
- High contrast for readability
- Use of icons/illustrations for test equipment
- Photography: Real tools, workbenches, technicians at work

## Canva Setup Instructions

### 1. Create Brand Kit
- Add all brand colors
- Upload UNI-T logo (get from kinmo.com)
- Upload Kinmo logo
- Set brand fonts

### 2. Create Templates
- Start with Facebook template (1200x630)
- Duplicate and resize for other platforms
- Use Canva's "Resize" feature for quick variants

### 3. Recommended Canva Elements
- Search "electrical" for relevant icons
- Search "multimeter" for product images
- Use "Frames" for product photo placement
- Use "Grids" for comparison layouts

### 4. Export Settings
- PNG for graphics with text
- JPG for photos (smaller file size)
- Export at 1x for web use
